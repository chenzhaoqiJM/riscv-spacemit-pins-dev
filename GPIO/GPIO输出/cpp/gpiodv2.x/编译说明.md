
## 依赖库:
sudo apt install libgpiod-dev

## 编译命令1

g++ led_gpiod_2x.cpp -o led_gpiod_2x -lgpiodcxx -lgpiod

自动计算chip index 和 引脚偏移

# 编译命令2

g++ led_gpiod_2x_hand.cpp -o led_gpiod_2x_hand -lgpiodcxx -lgpiod

# 运行（手动指定 chip 和 offset）

sudo ./led_gpiod_2x_hand 2 19

输出: 使用 /dev/gpiochip2 引脚 19

## 解释：

libgpiod 从 2.x 版本起，提供 C++ 接口和 C 接口：

| 库名               | 接口语言 | 说明                           |
| ---------------- | ---- | ---------------------------- |
| `libgpiod.so`    | C    | C API，例如 `gpiod_chip_open()` |
| `libgpiodcxx.so` | C++  | C++ API，例如 `gpiod::chip`     |

注意顺序：C++ 库依赖 C 库，-lgpiodcxx 放前，-lgpiod 放后最安全
